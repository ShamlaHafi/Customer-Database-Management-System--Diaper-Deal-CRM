# Customer-Database-Management-System--Diaper-Deal-CRM
This project presents a complete Database Management System (DBMS) design for Diaper Deal, a small-scale Sri Lankan baby care and hygiene product business. The objective of this system is to replace manual spreadsheet-based data management with a structured Customer Relationship Management (CRM) database system.
